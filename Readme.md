# 🍷 Vinharia Agnello

## 📖 Descrição do Projeto
Este projeto foi desenvolvido como parte de uma avaliação acadêmica da FIAP, com o objetivo de criar um site para a Vinharia Agnello.

A Vinharia Agnello é uma empresa familiar que atua há mais de 15 anos no mercado de vinhos, oferecendo uma grande variedade de rótulos nacionais e internacionais. Seu principal diferencial é o atendimento personalizado, auxiliando clientes na escolha de vinhos e harmonizações.

O projeto busca representar essa experiência no ambiente digital, por meio de um site organizado, informativo e com visual sofisticado.

O site é composto pelas seguintes páginas:

- **Index (Página Principal)**  
  Página inicial com apresentação sobre os vinhos e dicas.

- **Cadastro**  
  Página destinada ao cadastro de usuários.

- **Vinhos / Venda**  
  Página com exibição dos produtos disponíveis, incluindo informações como nome, preço e imagem.

- **Time de Vinheiros**  
  Apresentação da equipe da vinheria, com descrição dos membros e suas funções.

- **Sobre Nós**  
  Página com a história da vinheria, seus valores e principais características.
